1.4 减少网页中的HTTP请求

减少网页中的HTTP请求.md

网页-多个组件

一般一个网页都包含了多个组件,如html css js 和图片等等,这些都需要从服务端下载然后在前段计算渲染

没和快之间的博弈

  • 设计简单的网页,速度快乐但是美感下降了
  • 使用精灵技术,将多个图片合并为一个文件避免多次请求服务器下载,渲染时使用背景图片的偏移
  • 合并js脚本和css样式表
  • 使用cache
